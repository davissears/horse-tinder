# horse-tinder

A tiny vanilla JS demo of "Tinder for horses".

## Run locally

Just open `index.html` in your browser.

## GitHub Pages preview

This repo now includes a GitHub Actions workflow that deploys the site to GitHub Pages whenever `main` is updated.

### One-time setup

1. In GitHub, go to **Settings → Pages**.
2. Under **Build and deployment**, set **Source** to **GitHub Actions**.
3. Push to `main` (or run the workflow manually from the **Actions** tab).

After the deploy job finishes, your preview URL will appear in the workflow output and on the repository Pages settings screen.
